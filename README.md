# Clinical-Diagnostic-Classification
Clinical Diagnostic Classification (Cancer Data) | Python, Scikit-Learn

Developed and compared Logistic Regression and K-Nearest Neighbors (KNN) models to classify patient biopsy results.

Optimized model selection by analyzing Confusion Matrix metrics, prioritizing higher sensitivity to minimize False Negatives in diagnostic predictions.

Performed feature scaling and outlier detection on clinical patient data to improve KNN distance-based accuracy.
